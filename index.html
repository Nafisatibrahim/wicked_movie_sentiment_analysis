Wicked Sentiment Analysis Project (2024)

Overview

Analyzing public sentiment toward Wicked (2024), a highly anticipated movie adaptation of the beloved Broadway musical, using Python. This project combines data scraping, cleaning, visualization, and sentiment analysis to uncover how audiences perceive the movie.

Objective

As a data scientist, I performed data scraping, cleaning, visualization, and sentiment analysis, while ensuring insights matched the theme and spirit of Wicked.

The objective of this project is to understand public opinion toward the movie "Wicked" that was released in 2024. The potential applications include:

  • Identifying key themes and sentiments in audience reviews
  • Visualizing audience reception through word clouds and sentiment analysis
  • Providing actionable insights for the movie's producers and marketing team

Project Workflow

The project workflow included the following steps:

  1. Collecting review data using the Google Custom Search API
  2. Processing and cleaning the data to extract meaningful insights
  3. Visualizing the data through bar chart, line chart, word clouds and sentiment analysis
  4. Providing actionable insights based on the analysis

Data Collection

  • Source: Google Custom Search API.
  • Dataset: Reviews scraped with queries categorized into various themes, including general reviews, audience reactions, and critical analysis.
  • Data Attributes:
    • Query: Search query used.
    • Category: Theme of the query (e.g., General Reviews, Audience Reactions).
    • Title: Title of the review/article.
    • Link: URL of the review/article.
    • Snippet: Short text snippet of the review.

Sentiment Analysis

Sentiment analysis is a powerful tool that allows us to understand how people feel about a particular topic, product, or event. In this project, we used sentiment analysis to analyze audience reviews of the movie "Wicked" and identify key themes and sentiments.

In this project, we used the TextBlob library to perform sentiment analysis on the review data. TextBlob is a Python library that provides a simple API for common natural language processing (NLP) tasks, such as part-of-speech tagging, noun phrase extraction, sentiment analysis, classification, translation, and more.

Visualization and Insights

The results of this project can help the movie's producers understand audience reception, identify strengths and weaknesses, and strategize marketing campaigns. It also serves as an example of how data science can be used for real-world storytelling.

Key visualizations and insights include:

  • Bar chart of review ratings
  • Line chart of review sentiment over time
  • Word cloud of review keywords
  • Sentiment analysis of review data
